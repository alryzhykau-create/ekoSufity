# Codex sprint 3: Images, accessibility, performance

Optimize images and UX details for EkoSufity.

## Scope

1. Inspect existing images and image usage.
2. Ensure all meaningful images have Polish `alt` text.
3. Ensure image components have dimensions or equivalent sizing.
4. Lazy-load non-hero images.
5. Mark hero image as priority/eager if the framework supports it.
6. Create or update image path mapping using `images` fields from `ekoSufityContent.json`.
7. Improve clickable CTAs:
   - phone link
   - WhatsApp link
   - contact form links
8. Verify no layout shifts from images where easy to fix.

## Important

Do not claim placeholder/stock/AI images are real completed projects. Use neutral wording if needed.

## Acceptance criteria

- Alt texts are present and descriptive.
- No keyword stuffing in alt texts.
- Image sizes are controlled.
- Mobile CTA elements remain usable.
- Build/lint pass or failures are documented.
