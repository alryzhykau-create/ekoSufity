# EkoSufity

One-page and content site for stretch ceiling installation services in Wroclaw and nearby locations.

## Requirements

- Node.js 18.17 or newer
- npm

## Local Run

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```

## Production Check

Build the project:

```bash
npm run build
```

Start the production server after a successful build:

```bash
npm run start
```

## Useful Scripts

```bash
npm run dev
npm run build
npm run start
npm run lint
```

Note: `npm run lint` may ask for initial Next.js ESLint setup in this project. The main technical verification is `npm run build`.
